Return to D2ArmorPicker 关于 D2AP 由 Mijago 开发 ，萨勒你玛芬尼一半#9836 汉化 D2ArmorPicker（简称 D2AP）是一个小型 web 应用，它可以帮助你快速找到达成目标属性所需的盔甲。

由于我经常玩突袭，因此我希望可以围绕异域护甲预先构建出多个 build 以便切换，这样子可以使我不至于成为火力战队的负担，这是我开发的初衷。

初版使用 Python 构建，几天之后，我使用 web 技术将它变成了 web 应用。 我的朋友们觉得很好用，但是它没有 Bungie 的 API 权限， 因此我开发了第一版的 D2AP，在得到了越来越多玩家的喜爱之后，我重写了它，就是 2.0 版本！

我一直在对 D2AP 更新迭代，但请注意，我只能用业余时间进行这些工作。欢迎在 Twitter 关注我， 我偶尔会在上面发布更新。如果 D2AP 帮到了你，请在 ko-fi.com 请我喝杯咖啡吧！ 我该怎样使用 D2AP？ 以下是使用步骤：
  1. 登录。你已经完成了！
  2. [可选]选中一件你要用的异域护甲，D2AP 会展示出仅包含它的护甲组合结果，你随时可以取消此选中
  3. [可选]选择额外的模组或者星象碎片来提供属性加成/惩罚，这很重要
  4. 选择你想要的属性，这是 D2AP 的核心功能。你只需要把恢复设置为 100，它就会为你显示所有 100 恢复的护甲组合。注意：D2AP 会自动添加护甲属性模组。
  5. 现在查看结果列表，在大多数设备上它应该在右侧，在小屏幕设备上，它可能会在属性设置的下方。表格列出了多个护甲组合，你可以根据自己的需要对表格进行排序。
