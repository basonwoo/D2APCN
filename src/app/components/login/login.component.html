Destiny 2 Armor Picker 配装工具，由 Mijago 制作，萨勒你玛芬尼一半#9836 进行汉化

此工具允许你从装备栏和保险库中选择最佳的护甲搭配以满足您的六维需求

要使用此工具，必须进行登录以从 Bungie API 获取你的护甲。

如果你在使用过程中发现 bug 或者有更好的建议，请前往 Github 提 issue
如果你在使用过程中发现汉化错误，请前往 Github 提 issue

D2ArmorPicker 使用 Bungie 的 API 及其 OAuth 授权，用户在 Bungie 官网登录之后，D2ArmorPicker 会从 Bungie 接收到一个临时登录令牌。这意味着 D2ArmorPicker 并不会实际获取你的账户凭据。

示例 D2ArmorPicker 的功能 属性优化 核心功能

首先，你可以选择自己想要达成的属性数值，D2ArmorPicker 会自动分析你所有的护甲，然后展示出所有可能的组合结果。

你还可以预设好不同的星象碎片，此外，还可以选择 0 浪费模式，D2ArmorPicker 会自动添加属性模组以减少浪费。

展示结果 各种可选的护甲组合

D2ArmorPicker 会向你展示一个包含诸多护甲搭配的表格，你可以对表格进行排序 或者筛选，之后点击某一行来查看达成期望属性值所需要安装的模组。

护甲集合 整理保险库
自 2.0.14 版本开始，D2ArmorPicker 新增了护甲集合功能，这会将所有护甲根据六维分布分为 25 组， 每一组中的护甲都会有相似的六维属性，这将可以帮助你清理仓库。
